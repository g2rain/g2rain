# 行为准则（Code of Conduct）

为了营造一个友好、包容、专业的社区环境，g2rain 项目采用本行为准则。  
所有参与本项目的人员（包括使用者、贡献者、维护者等）都应遵守本准则。

本准则参考了 [Contributor Covenant](https://www.contributor-covenant.org/) 的精神并结合实际进行了简化调整。

---

## 1. 我们的承诺

我们致力于为每一位参与者提供一个：

- 不因年龄、性别、性别认同与表达、性取向、残疾、肤色、种族、国籍、宗教或技术背景而受到歧视的环境。
- 在其中每个人都能被尊重、愿意交流并乐于分享的社区。

---

## 2. 可接受的行为

在 g2rain 社区中，我们鼓励：

- 使用友好、包容的语言。
- 尊重不同的观点与经验。
- 积极、耐心地倾听他人的意见。
- 在给出反馈或评审时，聚焦问题本身，而非个人。
- 主动帮助新手与首次贡献者。

---

## 3. 不可接受的行为

以下行为在社区中是不可接受的，包括但不限于：

- 使用带有侮辱、攻击、歧视、骚扰意味的言语或图片。
- 人身攻击、贬低或羞辱他人。
- 故意引战、刷屏或恶意扰乱讨论秩序。
- 公布他人的隐私信息（真实姓名、联系方式、住址等）而未获得对方同意。
- 明知会对项目或用户造成严重风险，仍恶意提交代码或发布误导信息。

---

## 4. 适用范围

本行为准则适用于：

- g2rain 相关的所有 GitHub 仓库（包括 Issue、PR、Discussions 等）。
- 任何以 g2rain 名义组织的线上或线下活动。
- 其他社区交流渠道（如邮件列表、聊天群组）等。

---

## 5. 处理与执行

### 5.1 报告问题

如果你在社区中遇到不当行为，可以：

- 通过 GitHub 的举报功能（Report）标记相关内容。
- 通过邮件或私信的方式联系 Maintainer / Owner（如：`g2rain_developer@163.com`，需按实际配置）。

在报告时，请尽量提供：

- 相关链接（Issue、PR、评论等）。
- 发生时间、地点（线上 / 线下）。
- 涉及人员的 GitHub ID（如有）。

### 5.2 处理方式

维护团队在收到报告后，会根据情况采取以下措施之一或组合：

- 友善提醒或私下沟通，说明问题所在并建议改进。
- 编辑或删除不当内容（评论、Issue 等）。
- 暂时或永久禁止相关账号参与项目（Ban / Block）。
- 在极端情况下，向相关平台或机构报告。

具体措施将视问题严重程度与当事人态度而定。

---

## 6. 维护者责任

项目维护者应：

- 以身作则，率先遵守本行为准则。
- 在评审 PR、处理 Issue 和参与讨论时保持专业与尊重。
- 在出现冲突或误解时，尽可能基于事实和技术本身进行澄清。
- 在处理不当行为时，保持透明、公正与一致性。

---

## 7. 变更与反馈

本行为准则可能会根据社区发展的实际情况进行调整。  
你可以通过 Issue 或 PR 提出改进建议，帮助我们一起建设更健康的社区环境。

---

感谢你遵守本行为准则，并为 g2rain 社区营造积极、开放的氛围。
